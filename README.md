# 不動産投資分析レポート v9.0

AI搭載の不動産投資分析Webアプリ。物件情報を入力するだけで、キャッシュフロー・節税効果・長期シミュレーション・マイソクExcelを自動生成します。

## 機能
- AI物件分析（アピールポイント・市場分析）
- キャッシュvs節税効果（3段階）分析
- 長期シミュレーション（ローン完済まで）
- マイソク（販売図面）Excel出力
- 顧客開示モード

## 起動方法
```bash
pip install -r requirements.txt
python app.py
```

## 環境変数
- `OPENAI_API_KEY`: OpenAI APIキー（AI機能に必要）
